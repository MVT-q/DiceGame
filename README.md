# Dice Game

A cross-platform dice game built with .NET MAUI.

The project was created to practice the MVVM architecture, data binding, commands, and XAML in .NET MAUI.

## Features

- Single-player mode
- Two-player mode
- Configurable game settings
- Bonus game rules
- Roll history
- Dice rolling animation

## Implementation

- Dice faces are rendered programmatically instead of using image assets.
- The application follows the MVVM architectural pattern.

## Technologies

- C#
- .NET MAUI
- MVVM
- XAML

## How to run

1. Clone the repository.
2. Open the solution in Visual Studio.
3. Build and run the project.
